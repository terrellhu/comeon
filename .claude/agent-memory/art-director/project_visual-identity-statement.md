**Status:** Drafted, pending user approval before writing to art bible file.

**One-line visual rule:**
"Every frame must hold its weight as a sacred artifact of a ruined god — if it could hang in a corrupted temple, it belongs in the game."

**Principle 1 — Telegraph Is Texture** (serves: Read to Win)
Attack telegraphs are not UI overlays; they are integral compositional elements of the art itself. Glows, auras, charge pulses, silhouette distortions must be inseparable from the character's visual identity.
Design test: Choose the telegraph that, frozen as a still image, reads as the most compositionally complete illustration — not the most legible UI indicator.

**Principle 2 — One Boss, One Visual Grammar** (serves: Every Boss Is a Song)
Each boss owns a distinct color temperature, dominant shape language, and lighting direction that appears nowhere else in the game. No visual element may migrate between bosses.
Design test: If a particle, glow hue, or motion arc could appear in another boss's fight without looking wrong, redesign it until the answer is definitively no.

**Principle 3 — Corruption Is Visible, Glory Is Implied** (serves: Art Is Story)
Every fallen deity must simultaneously read as once-magnificent and now-broken. Both the divine grandeur layer and the corruption layer must survive a cover test independently.
Design test: Cover the corruption layer — is divine origin still readable? Cover the grandeur layer — is the tragedy still felt? Both must pass.

**Why this set works:** The one-liner and principles refuse the typical separation of art layer vs. UI/telegraph layer. This makes learning to read a boss identical to appreciating its art — the game's core differentiator.

**Why:** Finalized during initial art bible session. Grounded in ukiyo-e and religious icon tradition: maximum meaning per compositional unit, color used symbolically, deliberate silhouette.

**How to apply:** Reference these three tests whenever reviewing any boss visual asset, concept, or VFX proposal. If an asset fails any test, send back with specific corrective note citing which principle was violated.
