Game is called 刃响 (Blade Echo). A 2D boss-rush action game for PC (Steam).

Elevator pitch: Learn and exploit the visual rhythm of fallen deities. Each boss fight is a different song to master, presented through HD adult illustration aesthetics.

Core fantasy: You are the perfect counter-attacker. Victory comes from understanding the boss's visual language, not from stats or speed.

**Game Pillars:**
1. Read to Win — every parry has a clear visual telegraph; victory is visual understanding
2. Every Boss Is a Song — each boss has unique visual language, attack rhythm, narrative background
3. Art Is Story — HD adult aesthetics are not decoration; every frame tells the story of a fallen deity
4. Death Is a Teacher — quick retry, no loading, no penalty; failure costs time not frustration

**Visual identity anchor:** 「暗神话辉煌」(Dark Mythic Grandeur)
Visual rule: Every frame should feel like a hand-painted religious icon of a corrupted deity — beautiful, heavy, uncompromising.

**Reference games:** Hollow Knight, Bloodborne, Blasphemous, Furi
**Non-game references:** Japanese ukiyo-e, religious paintings, Princess Mononoke, Arcane

**Why:** Governs all visual decisions. The game's differentiation is that its readability system (telegraphs) and its aesthetic system (HD illustration) are not separated — they are the same thing.

**How to apply:** When making any visual proposal, verify it satisfies the sacred artifact framing. When a boss asset is ambiguous, check against the two-layer cover test (corruption visible, glory implied).
